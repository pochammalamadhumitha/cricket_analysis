# cricket_analysis
# 🏏 Cricket Analytics Project

## 📌 Overview
This project analyzes cricket (IPL) data to extract insights about player performance, team success, and match outcomes using Python and Machine Learning.

---

## 🎯 Objectives
- Analyze player performance (batting & bowling)
- Identify top-performing teams
- Visualize match trends
- Predict match outcomes using ML

---

## 🛠 Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

---

## 📊 Key Features

### ✅ Player Analysis
- Top run scorers
- Top wicket takers

### ✅ Team Analysis
- Most successful teams
- Winning trends

### ✅ Machine Learning
- Random Forest model to predict match winners
- Achieved accuracy: ~60–70%

---

## 📁 Dataset
- Source: Kaggle IPL Dataset
- Files used:
  - matches.csv
  - deliveries.csv
